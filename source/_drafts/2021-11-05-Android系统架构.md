# 版权

版权（Copyright）：也叫著作权。包括署名权、修改权、收益权等

侵犯版权：未经过版权拥有者许可，修改、发布作品等

创作作品的作者自动拥有版权，受版权法的保护，一般保护期限一般是100年。如果经过了修改，新作品会延长保护期。

```shell
# 指2000年创作，2018年经过了修改
Copyright 2000-2018 by XXX Ltd. All Rights Reserved
```

# 专利

专利也是一种知识产权，但和版权不同：

1. 专利是一种方法、或者一组方法的特征，版权是具体的作品或者实现
2. 专利需要申请，版权不需要申请
3. 专利申请后需要付费维持，版权不需要付费
4. 专利保护期限较短（例如10年、20年），版权保护期限更长（100年）

# GPL

GPL（GNU通用许可证）是一种开源许可协议

> GPL也称作Copyleft，是一种英语的调侃。Copyright（版权声明）中的right指权利。left表示相反。
>
> 实际上GPL也是一种Copyright，是软件作者对软件的授权。



Linux使用GPL协议，AOSP使用Apache协议。

Android为了规避GPL的传染性，在Linux Kernel的基础上搭建了HAL，把用户层和Kernel层分开，这样只有Kernel部分需要遵循GPL，用户层不需要，例如硬件驱动可以放在HAL层，不需要遵循GPL。
