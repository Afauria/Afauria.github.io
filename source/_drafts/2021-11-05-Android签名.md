# 签名工具

zipalign、apksigner、jarsigner、keytool



# v1、v2签名区别

V1对每个文件计算摘要，速度慢

V2对整个APK签名，不需要解压，对文件按1MB分块计算摘要，可以并行

https://zhuanlan.zhihu.com/p/108034286

https://blog.csdn.net/qq_32115439/article/details/55520012

jarsigner：V1签名，对压缩包中每个文件验证，签名之后可以解压修改压缩包

apksigner：V2签名，对整个压缩包文件验证，签名之后不能修改压缩包

zipalign：可以在V1之后执行，不能在V2之后执行

# AndroidStudio配置签名文件

```
android {
		signingConfigs {
        MySign {
            storeFile file("platform.keystore")
            storePassword "android"
            keyAlias "androiddebugkey"
            keyPassword "android"
        }
    }
    buildTypes {
        debug {
            signingConfig signingConfigs.MySign
        }
    }
}
```

# 系统签名

要获取系统权限，需要在`AndroidManifest.xml`中声明

```xml
<manifest
    xmlns:android="http://schemas.android.com/apk/res/android"
    package="xxx"
    android:sharedUserId="android.uid.system">
</manifest>
```

## 获取系统签名

获取系统的`platform.pk8`和`platform.x509.pem`：位于`android/build/target/product/security/`目录下

## 生成keystore

```shell
# pk8私钥解密成pem格式
openssl pkcs8 -in platform.pk8 -inform DER -outform PEM -out shared.priv.pem -nocrypt
# 私钥通过公钥pem加密pk12，输入两次密码
openssl pkcs12 -export -in platform.x509.pem -inkey shared.priv.pem -out shared.pk12 -name androiddebugkey
# JDK的keytool工具生成keystore
keytool -importkeystore -deststorepass $password -destkeypass $password -destkeystore platform.keystore -srckeystore shared.pk12 -srcstoretype PKCS12 -srcstorepass $password -alias $keyAlias
```

> 注：AndroidStudio中如果使用JDK8，keytool也要使用JDK8，否则编译会出现`Invalid keystore format`错误

## 使用signapk.jar

1. 下载`signapk.jar`：signapk源码位于：`android/build/tools/signapk`，编译完成后生成`android/prebuilts/sdk/tools/lib/signapk.jar`
2. `java -jar signapk.jar platform.x509.pem platform.pk8 $input $output`

## 使用jarsigner签名

`jarsigner -verbose -keystore $xxx.keystore -storepass $storePsd -signedjar $output $input old.apk $keyAlias`

## 使用apksigner.jar

1. 下载`apksigner.jar`：apksigner源码位于：`android/tools/apksig/src/apksigner`，编译完成后生成`android/prebuilts/sdk/tools/linux/lib/apksigner.jar`
2. `java -jar apksigner.jar sign --ks $xxx.keystore --ks-key-alias $keyAlias --ks-pass pass:$storePsw --key-pass pass:$storePsw --out $output $input`
