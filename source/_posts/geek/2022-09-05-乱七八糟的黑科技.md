# 前言

# Netsh

netsh（Network Shell）是Windows的命令行工具。参考[Windows netsh命令](https://docs.microsoft.com/zh-cn/windows-server/networking/technologies/netsh/netsh-interface-portproxy)

`netsh interface portproxy`代理IP地址和端口：貌似只能将外部进入本机的请求进行转发，无法对本机的出口进行转发

```shell
# 查看所有监听的端口
netsh interface portproxy show all
# 添加规则
netsh interface portproxy add v4tov4 listenaddress=监听IP地址 listenport=监听端口号 connectaddress=目标IP connectport=目标端口号 protocol=tcp
# 删除规则
netsh interface portproxy delete v4tov4 listenaddress=监听IP地址 listenport=监听端口号
```

# Iptables

Linux上常用的防火墙软件，很强大的功能。参考[iptables](https://blog.csdn.net/weixin_45186298/article/details/122910466)

可以进行入口、出口请求的转发、拦截等

```shell
# 查看规则
sudo iptables  -t nat --list OUTPUT
# -A添加规则
sudo iptables -t nat -A OUTPUT -d 47.99.96.61 -p tcp --dport 8008 -j DNAT --to-destination 127.0.0.1:80
# -D删除规则
sudo iptables -t nat -D OUTPUT -d 47.99.96.61 -p tcp --dport 8008 -j DNAT --to-destination 127.0.0.1:80
```
